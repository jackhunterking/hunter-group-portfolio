# Guides directory

Drop your final PDF files here with these exact filenames:

- `ev-alma-rehberi.pdf` — Buyer's guide (Alıcı)
- `ev-satma-rehberi.pdf` — Seller's guide (Satıcı)

The email template in `lib/email.ts` and the lead capture flow expect these paths.

If you rename them, update the `GUIDE_CONFIG` object in `lib/email.ts`.
